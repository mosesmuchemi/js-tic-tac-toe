# js-tic-tac-toe

A simple Tic-Tac-Toe game implemented in Javascript V8 3.14.5.9
Execute using NodeJS
Players 'X' and 'O' take turn inputing their position on the command line using numbers 1-9
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
